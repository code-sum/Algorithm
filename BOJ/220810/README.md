```python

            hap_cnt += 1
        elif input_[i + 2] == "(":

    print("none")

    if hap_cnt > sad_cnt:
        print("happy")
    elif hap_cnt < sad_cnt:
        print("sad")
    else:
        print("unsure")
```

#### 11724. ì—°ê²° ìš”ì†Œì˜ ê°œìˆ˜ [(link)](https://www.acmicpc.net/problem/11724)
